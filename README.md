# **Forensic Tool Combination**

\---

### Combines EvtxECmd + RECmd + PECmd

####   

| Tool | Description |  
| --------- | ---------------------------------------------------- |  
| EvtxECmd | Event log parser that provides human readable output |  
| RECmd | Registry hive searching |  
| PECmd | Analyzing prefetch files |  

## Dependencies:

1\. .NET Framework 6.0  
2\. Openpyxl  
3\. Ensure Python 3.12
